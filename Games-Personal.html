God Is Dead

Github Link

God Is Dead is an work-in-progress GB, GBC and GBA game I have began to create using GB Studio. It is an RPG, Boss Rush, Roguelike where God, creator of all life, has fallen. The creations, of this almighty being, fight in an tournament where the last one standing is crowned the new Creator of All. The player will take control of an unnamed entity capable of rebirth, who, like the rest of the creations aim, to become the new God.

Balloon Blaster

Github Link

Balloon Blaster is an work-in-progress arcade style game where you get points by shooting balloons as they descend the screen. It is being created in Visual Studio using SFML and C++. Pink balloons give two hundred points, Blue balloons give four hundred and gold ones give six hundred. Minus ballons will take two hundred points and multiply balloons will multiply the score by two. The game ends when the player shoots a skull balloon where they will be shown their score.
