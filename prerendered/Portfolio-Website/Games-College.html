Short Circuit

Codeberg Link

Short Circuit is a top-down web based game, created in Visual Code using HTML, CSS and JS. The player takes control of a robot that is damaged and needs to collect items to repair itself. Each wire or wrench adds five to the repair score. If the player colides with the bug, they lose five repair points. Depending on the repair score the player gets when the timer runs out, they will get one of three endings, a good end, neutral end or a bad end.
